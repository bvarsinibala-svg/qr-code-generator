# QR Code Generator

## 📌 Project Overview

QR Code Generator is a simple web application that allows users to generate QR codes instantly from any text, URL, email address, or other input. The application is built using HTML, CSS, and JavaScript.

## 🚀 Features

* Generate QR codes from text or URLs
* Simple and user-friendly interface
* Responsive design
* Instant QR code generation
* Works directly in the browser

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* QR Code API

## 📂 Project Structure

qr-code-generator/

├── index.html

├── style.css

├── script.js

└── README.md

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Open `index.html` in a browser or use Live Server.
4. Enter text or a URL.
5. Click **Generate QR Code**.
6. The QR code will be displayed instantly.

## 📸 Usage

* Enter any text, URL
* Click the Generate QR Code button.
* Scan the generated QR code using any QR scanner.
